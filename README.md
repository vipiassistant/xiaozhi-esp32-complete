# Xiaozhi ESP32 Complete Build

## Thông tin build
- **Version**: v2.0.2
- **ESP-IDF version**: v5.5
- **Target**: ESP32-S3
- **Flash size**: 16MB
- **PSRAM**: 8MB

## Tính năng
- Wi-Fi Access Point (ViPi-XXXX)
- Custom wake word (Vi Bi)
- Audio codec 16kHz/24kHz
- OLED display
- MCP server

## Cài đặt
1. Sử dụng ESP-IDF v5.5
2. Flash vào ESP32-S3
3. Kết nối Wi-Fi AP để cấu hình
